# IMPL Academic Management System

Sistema web para gerenciamento de base de dados académica desenvolvido para o Instituto Médio Politécnico da Lunda Sul (IMPL).

## 📌 Descrição

Este sistema tem como objetivo informatizar e otimizar a gestão académica da instituição, permitindo o controle eficiente de:

- Alunos
- Professores
- Matrículas
- Cursos
- Turmas e classes
- Propinas e pagamentos
- Relatórios académicos

A aplicação foi desenvolvida com foco em organização, escalabilidade e facilidade de uso, substituindo processos manuais por uma solução digital integrada.

---

## ⚙️ Tecnologias Utilizadas

- **Laravel** (Backend)
- **Blade** (Frontend)
- **TailwindCSS** (Estilização)
- **Vite** (Build tool)
- **MySQL** (Base de dados)

---

## 🧱 Arquitetura

O sistema segue uma arquitetura organizada baseada em:

- Controllers
- Services (lógica de negócio)
- Models (Eloquent ORM)
- Form Requests (validação)

---

## 🚀 Funcionalidades

- Autenticação de utilizadores
- Gestão de perfis (Admin / Secretaria)
- Gestão de alunos
- Gestão de professores
- Gestão de cursos, classes e turmas
- Matrículas
- Controle de propinas e pagamentos
- Geração de relatórios em PDF

---

## 🎯 Objetivo

Desenvolver uma solução académica eficiente que possa ser utilizada inicialmente no IMPL e posteriormente adaptada para outras instituições de ensino.
